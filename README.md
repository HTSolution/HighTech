# 🚀 HighTech | Portfolio Experience

Un'esperienza web d'avanguardia realizzata con **Next.js 14**, **Framer Motion** e **Tailwind CSS**. 
Il sito presenta un'interfaccia ultra-fluida, navigazione "smart" e transizioni di pagina immersive stile mobile-app.

## 🛠 Tech Stack

* **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
* **Animazioni**: [Framer Motion](https://www.framer.com/motion/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Componenti**: React Functional Components (use client)
